Описание проекта
Random Password Generator — это десктопное приложение с графическим интерфейсом для генерации криптостойких случайных паролей. Программа позволяет гибко настраивать параметры пароля, оценивать его сложность и сохранять историю генерации.

🎯 Цели проекта
🔐 Генерация надёжных случайных паролей

⚙️ Гибкая настройка параметров (длина, типы символов)

📊 Оценка сложности сгенерированных паролей

📜 Сохранение истории генерации

📋 Быстрое копирование в буфер обмена

💾 Локальное хранение данных в JSON

👥 Целевая аудитория
Категория	Использование
IT-специалисты	Генерация паролей для серверов и сервисов
Обычные пользователи	Создание надёжных паролей для аккаунтов
Системные администраторы	Генерация временных паролей для сотрудников
Студенты	Изучение криптографии и генерации случайных чисел
Семейное использование	Создание безопасных паролей для всех членов семьи
Безопасники	Проверка стойкости парольных политик
🌟 Ключевые особенности
Особенность	Описание
✅ Криптостойкость	Использование модуля secrets для генерации
✅ Настраиваемая длина	От 4 до 128 символов с визуальным ползунком
✅ 4 типа символов	Цифры, буквы, заглавные, спецсимволы
✅ Оценка сложности	4 уровня: Слабый → Очень сильный
✅ История паролей	Сохранение всех сгенерированных паролей
✅ Копирование в буфер	Один клик для копирования
✅ JSON хранение	Автосохранение и загрузка истории
✅ Оффлайн работа	Полностью автономное приложение
