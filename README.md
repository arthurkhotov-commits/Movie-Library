 📝 Описание

**Movie Library** — это десктопное приложение с графическим интерфейсом для управления личной коллекцией фильмов. Приложение позволяет пользователям вести каталог просмотренных или планируемых к просмотру фильмов с возможностью фильтрации, поиска и анализа коллекции.

🎯 Основные возможности

- ✅ **Добавление фильмов** с указанием названия, жанра, года выпуска и рейтинга
- 📋 **Просмотр коллекции** в удобной таблице с чередованием строк
- 🔍 **Фильтрация** по жанру (частичное совпадение) и году выпуска (точное совпадение)
- 📊 **Статистика** коллекции: средний рейтинг, распределение по жанрам, временной диапазон
- 💾 **Сохранение данных** в формате JSON с автоматическим backup
- 📂 **Загрузка данных** из JSON файла
- 🗑️ **Удаление фильмов** с подтверждением
- ✅ **Валидация ввода** с подробными сообщениями об ошибках
- 🔄 **Проверка дубликатов** при добавлении
- 📌 **Статусная строка** с информацией о текущем состоянии

 🚀 Установка и запуск
 Нажмите зеленую кнопку Code → Download ZIP

Распакуйте архив в любую папку

Запустите movie_library.py

🖥️ Запуск в Thonny IDE
Откройте Thonny

Нажмите File → Open и выберите movie_library.py

Нажмите зеленую кнопку ▶️ Run (или F5)

📋 Требования

- **Python** 3.7 или выше
- **Tkinter** (входит в стандартную библиотеку Python)
- **Операционная система**: Windows, Linux, macOS
