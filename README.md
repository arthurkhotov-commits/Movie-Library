Random Task Generator
Автор: Токумаева Дарина Магомедовна


Описание проекта
Random Task Generator — это десктопное приложение с графическим интерфейсом, предназначенное для генерации случайных задач и ведения их истории. Приложение помогает пользователям разнообразить свою повседневную деятельность, случайным образом выбирая задачу из предустановленного или пользовательского списка.

Программа особенно полезна для:

Людей, которым сложно выбрать, чем заняться

Формирования привычек и рутины

Тайм-менеджмента и борьбы с прокрастинацией

Геймификации повседневных дел

Функциональные возможности
Основной функционал
Функция	Описание
🎲 Генерация случайной задачи	Выбирает случайную задачу из общего пула (предопределённые + пользовательские)
📜 История задач	Отображает все сгенерированные задачи в хронологическом порядке
🔍 Фильтрация	Позволяет фильтровать историю по типу задачи: учёба, спорт, работа
➕ Добавление задач	Пользователь может добавлять свои задачи с указанием типа
💾 Сохранение истории	Автоматическое сохранение истории в JSON-файл
🔄 Загрузка истории	При запуске история загружается из файла
🗑 Очистка истории	Возможность полной очистки истории с подтверждением
Валидация данных
При добавлении новой задачи проверяется, что поле описания не пустое

Тип задачи обязательно должен быть выбран из списка

Ошибочные действия сопровождаются всплывающими предупреждениями
