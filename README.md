Описание проекта
Training Planner — это десктопное приложение с графическим интерфейсом для планирования, учёта и анализа тренировок. Программа помогает пользователям вести дневник физической активности, отслеживать прогресс и формировать здоровые привычки.

🎯 Цели проекта
📝 Планирование - создание расписания тренировок

📊 Учёт - фиксация выполненных тренировок

📈 Анализ - статистика и прогресс активности

💾 Сохранность - локальное хранение данных

🎯 Мотивация - визуализация достижений

👥 Целевая аудитория
Категория	Использование
Спортсмены	Профессиональный учёт нагрузок
Фитнес-энтузиасты	Отслеживание прогресса
Новички	Формирование привычки тренироваться
Тренеры	Контроль подопечных
ЗОЖ-активисты	Ведение дневника активности
Реабилитологи	Контроль восстановления
🌟 Ключевые особенности
✅ 10 типов тренировок на любой вкус

✅ Двойная фильтрация (по типу и месяцу)

✅ Автоматическая статистика в реальном времени

✅ Валидация всех полей ввода

✅ Тёмная тема для комфортной работы

✅ JSON хранение без потери данных
