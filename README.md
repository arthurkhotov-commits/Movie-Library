Описание проекта
Random Quote Generator — это десктопное приложение с графическим интерфейсом для генерации вдохновляющих, мотивирующих и философских цитат. Программа случайным образом выбирает цитату из встроенной коллекции или из цитат, добавленных пользователем, сохраняет историю просмотров и позволяет фильтровать содержимое по автору и теме.

Функциональные возможности
✨ Основной функционал
№	Функция	Описание	Статус
1	Генерация случайной цитаты	Выбор цитаты из предопределённых и пользовательских	✅
2	Отображение текущей цитаты	Показ текста, автора и темы в удобном формате	✅
3	История просмотров	Хранение всех сгенерированных цитат в хронологическом порядке	✅
4	Фильтрация по автору	Отображение цитат только выбранного автора	✅
5	Фильтрация по теме	Отображение цитат только выбранной тематики	✅
6	Сброс фильтров	Возврат к полной истории одним кликом	✅
7	Добавление цитат	Пополнение базы своими цитатами	✅
8	Очистка истории	Удаление всей истории с подтверждением	✅
💾 Работа с данными
Возможность	Описание

Пустое имя автора	Предупреждение, отказ в добавлении
Невыбранная тема	Предупреждение, отказ в добавлении
Повреждённый JSON-файл	Создание новой чистой истории

Установка и запуск
📥 Способы установки
Способ 1: Запуск из исходного кода (рекомендуемый)
Создайте файл random_quote_generator.py и скопируйте код из репозитория

Убедитесь, что Python установлен:

bash
python --version
# Должно быть Python 3.7 или выше
Запустите приложение:

bash
python random_quote_generator.py
