Описание проекта
GitHub User Finder — это десктопное приложение с графическим интерфейсом для поиска и отслеживания пользователей GitHub. Приложение использует официальное GitHub API для получения информации о разработчиках и позволяет сохранять интересных пользователей в избранное.

🎯 Цели проекта
Предоставить удобный интерфейс для поиска пользователей GitHub

Отображать детальную информацию о разработчиках (био, репозитории, подписчики)

Позволять сохранять избранных пользователей для быстрого доступа

Сохранять данные локально в JSON формате

Демонстрировать работу с REST API в Python

👥 Целевая аудитория
Разработчики - поиск коллег и интересных проектов

HR-специалисты - поиск талантов для найма

Open Source энтузиасты - отслеживание контрибьюторов

Студенты - изучение GitHub и API

Рекрутеры - формирование базы кандидатов

🌟 Особенности
✅ Без установки дополнительных библиотек - работает только на встроенных модулях Python

✅ Асинхронные запросы - интерфейс не зависает при загрузке

✅ Локальное сохранение - избранное хранится в JSON файле

✅ Обработка ошибок - корректная работа при отсутствии интернета

Функциональные возможности
✨ Основной функционал
№	Функция	Описание	Статус
1	Поиск пользователей	Поиск по username на GitHub	✅
2	Детальная информация	Отображение имени, био, статистики	✅

📊 Отображаемая информация
Поле	Описание
👤 Имя	Полное имя пользователя
🔑 Логин	GitHub username
💬 Биография	Описание профиля (до 200 символов)
📁 Репозитории	Количество публичных репозиториев
👥 Подписчики	Количество подписчиков
📢 Подписки	Количество подписок
📅 Дата добавления	Время добавления в избранное
🛡️ Валидация
Проверка	Действие
Пустое поле поиска	Предупреждение, поиск не выполняется
Пользователь не найден (404)	Сообщение об ошибке
Нет интернета	Сообщение об ошибке
Таймаут запроса	Сообщение об ошибке
